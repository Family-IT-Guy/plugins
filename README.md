# Family IT Guy Plugins

Claude Code plugin marketplace for the FITG team. Private plugins, team access only.

## Install

```bash
# Add the marketplace (one-time)
/plugin marketplace add Family-IT-Guy/fitg-plugins

# List available plugins
/plugin list

# Install a plugin
/plugin install fitg-analytics@fitg
```

Restart Claude Code after installing.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `fitg-analytics` | Product analytics across Stripe, Fathom, CloudWatch, PostHog, and Meta. `/analytics` command. |

## Access

All source repos are private. You need GitHub access to `Family-IT-Guy` to install plugins.
