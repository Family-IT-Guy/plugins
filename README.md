# Family IT Guy Plugins

Claude Code plugin marketplace by Family IT Guy.

## Install

```bash
# Add the marketplace (one-time)
claude plugins marketplace add Family-IT-Guy/plugins

# Install a plugin
claude plugins install research-engine@familyitguy
```

Restart Claude Code after installing.

## Available Plugins

| Plugin | Description | Public |
|--------|-------------|--------|
| `research-engine` | Cited web research via Perplexity Sonar API. Background sub-agents, primary source verification, cascade follow-ups. | Yes |
| `fitg-analytics` | Product analytics across Stripe, Fathom, CloudWatch, PostHog, and Meta. | No |
